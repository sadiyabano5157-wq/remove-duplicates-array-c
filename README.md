# Remove Duplicates from Array (C Program)

## 📌 Description
This program removes duplicate elements from an array and stores only unique elements in a new array.

## 🚀 How It Works
- Traverse the original array.
- Check if the current element already exists in the new array.
- If it does not exist, add it to the new array.
- Print the new array containing unique elements only.



## 📥 Input
```text
10 20 10 30 20
```

## 📤 Output
```text
Array after removing duplicates:
10 20 30
```

- Arrays
- Nested Loops
- Duplicate Detection
- Array Traversal
